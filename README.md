# Atividade-Pratica-Coleta-de-Clima-Open-Meteo-Banco-de-Dados-Python-
Atividade Prática — Coleta de Clima com Open-Meteo e Banco de Dados em Python

Este projeto em Python consome dados climáticos da API Open-Meteo e salva as medições em um banco de dados usando SQLAlchemy.
Ele permite coletar informações como temperatura, umidade relativa e código de condição climática em tempo real e registrar no banco para consultas futuras.

Funcionalidades: 
Consumir API de clima: Obtém informações meteorológicas atuais pela API gratuita Open-Meteo.
Persistência em banco de dados: Salva os dados em uma tabela Clima usando SQLAlchemy.
Tratamento de erros para evitar registros inválidos no banco.
Estrutura simples em Python, fácil de adaptar para diferentes localidades.
